The Obligatory Bio.

Making computers do stuff is fun. Learning how they do it is even better. Back in the day, I built Angelfire sites with my friends. We had no idea what we were doing, but we had fun. All these years later, writing code and building software is still just as fun, except now I actually know how to do it.

Originally from Boston, Massachusetts, I moved to Nashville after having my fill of New England winter in late 2011. Once I got here, I started working for an HR Consulting company. My interest in tech was ever present, and the idea of learning to code was always in the back of my mind. In late 2016 I found out about Nashville Software School and finally took the plunge and enrolled in April, 2017. After the intense six month course, I'm ready to take the next step become the best developer I can be. The best part is there's always going to be something new to learn. And I like learning stuff.

When I'm not writing code or learning a new aspect of programming, I'm usually reading a book, cooking, hiking, or playing with my dogs. I also like trying new craft beers, and I usually do this while engaged in another hobby. Its important to be able to multitask.

Some Projects I've Worked On.

DogFetcher

A single page web application built to help users find rescue dogs in their local area. Using the PetFinder.com API, a random dog is displayed upon page load. The user is able to save the dog to their profile, view a listing of twenty five local shelters in the Nashville, TN area, as well as the complete listings for each shelter. Built with AngularJS, Sass, Bootstrap and Firebase.

BiblioFile

BiblioFile is an MVC web app that allows uses to catalogue what’s on their bookshelf, both what has been read and what the user is planning on reading. The user’s collections are searchable and books can be added or removed at any time. User’s books can also be moved from the “To Read” section to the “Read” table. Built using C#/.NET, ASP.NET, SQLite and Bootstrap.
